1. PROJECT DESCRIPTION
===========================================================
This is a responsive and animated static website for Vinegrape Academy, a primary school based in Ruiru, KIhunguro. The goal is to present the school professionally to both parents and learners with an engaging, easy-to-navigate, and visually appealing interface.

No backend logic is implemented — all pages are static and purely informational. Content such as images, logo, and video clips will be inserted later when available. The backend will be uploade later.
